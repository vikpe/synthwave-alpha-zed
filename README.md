> A Synthwave inspired dark theme for Zed

## Installation

### Via Zed

1. Open extensions
2. Search for `Synthwave Alpha`
3. Click install
4. Enable in theme selector

### Manually

1. Download [`synthwave-alpha.json`](./themes/synthwave-alpha.json) to `~/.config/zed/themes`
2. Enable in theme selector
